# 🎨 Portfolio Website

A modern, interactive portfolio website built with React, TypeScript, and cutting-edge web technologies. Features smooth animations, responsive design, and a dark/light theme toggle.

## ✨ Features

- **Smooth Animations**: Framer Motion for fluid transitions
- **Dark/Light Theme**: Built-in theme provider for user preference
- **Responsive Design**: Tailwind CSS for mobile-first styling
- **Smooth Scrolling**: Custom scroll behavior with Lenis
- **Fast Performance**: Built with Vite for lightning-fast development and builds
- **TypeScript Support**: Fully typed for better development experience

## 🚀 Tech Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Animations**: 
  - Framer Motion
- **UI Components**: Lucide React, React Icons
- **Smooth Scrolling**: Lenis
- **Linting**: ESLint

## 🛠️ Installation & Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## 📝 Available Scripts

- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production
- `npm run lint` - Run ESLint checks
- `npm run preview` - Preview production build
